Freepik Icons API Demo

Search and use PNG icons dynamically in visualizations

How it works

  • PNG thumbnails (128, 256, 512px) are free to use from search results
  • SVG downloads cost 150 credits each
  • For visualizations, 512px PNGs work great and cost nothing!
  • Use freepik.searchIcons('term') to search, then grab icon.thumbnails[2].url for 512px

Usage in Your Visualizations

// Include the API utility
<script src="freepik-api.js"></script>

// Initialize
const freepik = new FreepikAPI('YOUR_API_KEY');

// Search and get icon PNG for a D3 visualization
const result = await freepik.searchAndGetIcon('cooking');
console.log(result.pngUrl); // 512px PNG URL
// Use in D3 force diagram
const icons = await freepik.batchSearch(['pasta', 'tomato', 'cheese']);

node.append('image')
  .attr('href', d => icons.get(d.name)?.pngUrl)
  .attr('width', 40)
  .attr('height', 40);
// For recipe visualization - preload ingredient icons
const ingredients = ['egg', 'bacon', 'pepper', 'salt'];
const iconMap = await freepik.batchSearch(ingredients);

// Add to your boxes
ingredients.forEach(name => {
  const icon = iconMap.get(name);
  if (icon) {
    document.querySelector(`#\${name}-box img`).src = icon.pngUrl;
  }
});
